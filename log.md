# Project Update Log

## [2026-03-09] Phase 1: JJSTAR Guide 콘텐츠 표준화 및 레이아웃 강화

### 작업 요약
JJSTAR Guide의 시각적 가독성을 높이고 일관된 UI를 유지하기 위한 디자인 시스템 구축 및 뷰어 기능 고도화를 완료했습니다.

### 상세 변경 내역

#### 1. 디자인 시스템 구축 (`assets/css/style.css`)
- **Typography**: `Pretendard` 서체 적용 및 본문 가독성 최적화 (Line-height 1.65, Font-size 16px).
- **Callout**: 보충 설명(info), 주의(warning), 팁(tip)을 위한 강조 박스 컴포넌트 추가.
- **Step Card**: Grid 레이아웃을 활용한 단계별 안내 카드 UI 구현 (Hover 효과 포함).
- **Table**: 메뉴 경로 및 기능 설명을 위한 세련된 표 스타일 정의.
- **Media**: 이미지 테두리, 그림자, 캡션 스타일 및 확대(Zoom-in) 커서 적용.

#### 2. 메인 뷰어 고도화 (`index.html`)
- **Layout**: 콘텐츠 영역 최대 너비를 1000px(본문 900px)로 제한하고 중앙 정렬하여 시선 분산 방지.
- **Breadcrumb**: 페이지 로드 시 `대분류 > 중분류 > 소분류`가 자동으로 조합되어 표시되도록 로직 개선.
- **Lightbox**: 별도 라이브러리 없이 Vanilla JS로 이미지 클릭 시 확대 기능 구현 (배경 흐림 효과 및 닫기 기능 포함).
- **UX**: 문서 로딩 시 최상단으로 자동 스크롤 기능 추가.

#### 3. 표준 템플릿 제작 (`pages/template.html`)
- 향후 제작될 모든 가이드 문서의 기준이 되는 HTML 템플릿 생성.
- 모든 디자인 시스템 컴포넌트(Callout, Step Card, Table, Media)의 샘플 코드를 포함하여 복사-붙여넣기 사용 가능하도록 구성.

## [2026-03-09] Phase 2: 지능형 검색 및 내비게이션 고도화

### 작업 요약
방대한 메뉴 구조에서 원하는 정보를 즉시 찾고 현재 위치를 직관적으로 파악할 수 있도록 검색 엔진 및 내비게이션 UX를 대폭 강화했습니다.

### 상세 변경 내역

#### 1. 지능형 초성 검색 엔진 구현 (`assets/js/search.js`)
- **초성 검색**: 한글 초성(ㄱㄴㄷ...)만으로 메뉴를 검색할 수 있는 알고리즘을 도입하여 접근성을 높였습니다. (예: 'ㅅㅈㄷㅎ' -> '사제동행교수선택')
- **실시간 필터링**: 입력 시 즉시 `JJSTAR_MENU` 데이터를 필터링하여 사이드바를 업데이트합니다.
- **키워드 하이라이팅**: 검색어와 일치하는 텍스트에 `<mark>` 태그를 적용하여 시각적으로 강조했습니다.

#### 2. 사이드바 내비게이션 UX 강화 (`index.html`, `style.css`)
- **Active 상태 동기화**: 현재 읽고 있는 문서에 해당하는 메뉴 항목에 `.is-active` 스타일(배경색 및 강조선)을 실시간으로 적용합니다.
- **자동 아코디언**: 문서 로드 시 해당 항목이 속한 상위 카테고리 및 중분류가 자동으로 펼쳐지도록 개선했습니다.
- **Auto-Scroll**: 선택된 메뉴 항목이 사이드바에서 보이지 않을 경우, 자동으로 화면 중앙으로 스크롤(`scrollIntoView`)되도록 구현했습니다.
- **검색 결과 카운팅**: 검색창 하단에 현재 필터링된 메뉴의 개수를 실시간으로 표시합니다.

#### 3. 동적 경로 추적 및 헤더 동기화
- **Dynamic Breadcrumb**: 메뉴 데이터를 역추적하여 현재 위치를 `대분류 > 중분류 > 소분류` 형태로 정확하게 표시합니다.
- **제목 자동 업데이트**: 문서 로드 시 해당 메뉴의 제목이 본문 H1 영역에 자동으로 반영되도록 구현했습니다.
