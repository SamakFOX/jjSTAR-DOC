# Project Update Log

## [2026-03-09] Phase 1: JJSTAR Guide 콘텐츠 표준화 및 레이아웃 강화

### 작업 요약
JJSTAR Guide의 시각적 가독성을 높이고 일관된 UI를 유지하기 위한 디자인 시스템 구축 및 뷰어 기능 고도화를 완료했습니다.

### 상세 변경 내역

#### 1. 디자인 시스템 구축 (`assets/css/style.css`)
- **Typography**: `Pretendard` 서체 적용 및 본문 가독성 최적화 (Line-height 1.65, Font-size 16px).
- **Callout**: 보충 설명(info), 주의(warning), 팁(tip)을 위한 강조 박스 컴포넌트 추가.
- **Step Card**: Grid 레이아웃을 활용한 단계별 안내 카드 UI 구현 (Hover 효과 포함).
- **Table**: 메뉴 경로 및 기능 설명을 위한 세련된 표 스타일 정의.
- **Media**: 이미지 테두리, 그림자, 캡션 스타일 및 확대(Zoom-in) 커서 적용.

#### 2. 메인 뷰어 고도화 (`index.html`)
- **Layout**: 콘텐츠 영역 최대 너비를 1000px(본문 900px)로 제한하고 중앙 정렬하여 시선 분산 방지.
- **Breadcrumb**: 페이지 로드 시 `대분류 > 중분류 > 소분류`가 자동으로 조합되어 표시되도록 로직 개선.
- **Lightbox**: 별도 라이브러리 없이 Vanilla JS로 이미지 클릭 시 확대 기능 구현 (배경 흐림 효과 및 닫기 기능 포함).
- **UX**: 문서 로딩 시 최상단으로 자동 스크롤 기능 추가.

#### 3. 표준 템플릿 제작 (`pages/template.html`)
- 향후 제작될 모든 가이드 문서의 기준이 되는 HTML 템플릿 생성.
- 모든 디자인 시스템 컴포넌트(Callout, Step Card, Table, Media)의 샘플 코드를 포함하여 복사-붙여넣기 사용 가능하도록 구성.
