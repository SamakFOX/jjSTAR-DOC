# Phase 3-2: 상세 가이드 페이지 템플릿 최적화 (장학금 수혜 이력)

---

## 1. 프로젝트 개요
- **목표**: 5장의 스텝별 이미지를 활용하여 '장학금 수혜 이력 조회' 가이드를 토스/애플 스타일의 미니멀한 레이아웃으로 구현
- **구조**: [메인 타이틀] -> [바로가기 버튼] -> [기능 개요] -> [상세 스텝별 설명(이미지 포함)]
- **핵심 원칙**: 1. 2. 3. 과 같은 번호 체계 제거, 여백 강조, 중앙 관리형 링크 시스템 적용

---

## 2. 세부 수행 단계

### [Step 1] 상세 페이지 전용 모던 CSS 적용
`assets/css/style.css`에 다음 스타일을 추가하거나 업데이트하세요.
1.  **Layout**: 본문 최대 너비를 `840px`로 제한하고 좌우 여백을 충분히 확보합니다.
2.  **Typography**: 
    - 메인 제목(`h1`): `36px`, Bold, 자간 축소.
    - 소제목(`h3`): `24px`, Margin-top `60px`. (번호 없이 제목만으로 섹션 구분)
    - 본문(`p`): `18px`, Line-height `1.8`, 색상 `#4e5968`(Toss 스타일 회색).
3.  **Quick Link Button**:
    - `.btn-direct`: 배경색 `#005da8`, 흰색 글자, 둥근 모서리(`12px`), 넉넉한 패딩.
    - 버튼 내부에 `↗` 아이콘을 추가하여 외부 링크임을 명시합니다.
4.  **Image Shadow**: 이미지는 테두리 대신 부드러운 그림자(`box-shadow: 0 4px 20px rgba(0,0,0,0.08)`)를 적용합니다.

### [Step 2] 상세 페이지(janghak-history.html) 구조 재작성
제공된 이미지 5장과 설명을 바탕으로 다음 구조를 생성하세요.

1.  **헤더**: 제목과 요약 설명.
2.  **바로가기**: `id="direct-link"`를 가진 버튼 영역.
3.  **본문 1 (전체 개요)**: 1번 이미지를 활용하여 학년/학기 설정(1), 리스트(2), 합계(3)의 위치를 안내합니다.
4.  **본문 2 (상세 스텝)**: 2번~5번 이미지를 활용하여 순차적으로 설명합니다. (번호 생략)

### [Step 3] 중앙 관리형 링크 시스템 (JavaScript)
`app.js` 또는 `index.html` 하단 스크립트에 다음 로직을 반영하세요.
1.  **URL 데이터셋**: 
    ```javascript
    const GUIDE_LINKS = {
      "c01_m02_s01": "[https://jjstar.jbnu.ac.kr/main/scholarship/history](https://jjstar.jbnu.ac.kr/main/scholarship/history)", // 예시 URL
      // 나머지 메뉴 코드와 URL 매핑
    };
    ```
2.  **링크 주입**: 가이드가 로드될 때 현재 페이지의 코드를 인식하여 버튼의 `href`를 자동으로 변경합니다.

## 3. 콘텐츠 가이드 (내용 구성)

- **섹션 1: 장학금 수혜현황 한눈에 보기** (이미지 1번)
  - "장학금 조회 화면은 크게 세 영역으로 나뉩니다. 상단에서 조건을 설정하고, 중앙에서 내역을 확인하며, 하단에서 총액을 볼 수 있습니다."
- **섹션 2: 조회 조건 입력** (이미지 2, 3번)
  - "확인하고자 하는 학년도와 학기를 선택하세요. 전체 내역이 궁금하다면 학기를 '전체'로 설정한 뒤 조회 버튼을 클릭합니다."
- **섹션 3: 수혜 내역 및 합계 확인** (이미지 4, 5번)
  - "선택한 조건에 맞는 장학금 명칭과 지급 금액이 리스트로 출력됩니다. 모든 수혜 금액의 총합은 화면 최하단에서 확인 가능합니다."

## 4. 검증 포인트 (Checklist)

- [ ] **디자인**: 번호(1. 2.)가 없어도 제목의 크기와 간격만으로 단계가 자연스럽게 읽히는가?
- [ ] **링크**: 바로가기 버튼이 정상적으로 작동하며, 중앙 관리 스크립트가 링크를 잘 주입하는가?
- [ ] **이미지**: 5장의 이미지가 설명 텍스트와 적절한 간격을 두고 배치되었는가?
