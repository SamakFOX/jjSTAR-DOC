# Phase 1: JJSTAR Guide 콘텐츠 표준화 및 레이아웃 강화

---

## 1. 프로젝트 개요
- **목표**: JJSTAR Guide의 시각적 가독성을 극대화하고, 수백 개의 가이드 페이지가 일관된 UI를 유지하도록 디자인 시스템 구축
- **핵심 작업**: 가이드 전용 CSS 컴포넌트 제작, 메인 뷰어 레이아웃 개선, 표준 HTML 템플릿 생성
- **기술 제약**: HTML5, CSS3, Vanilla JS만 사용 (외부 라이브러리/프레임워크 사용 금지)

---

## 2. 세부 수행 단계

### [Step 1] 가이드 전용 디자인 시스템(CSS) 구축
`assets/css/style.css`에 다음의 전문 가이드용 클래스를 설계하고 추가하세요.

1.  **Callout (강조 박스)**:
    - `.guide-callout.info`, `.guide-callout.warning`, `.guide-callout.tip` 구분
    - 배경색, 좌측 테두리 포인트, 아이콘(Emoji)이 포함된 눈에 띄는 디자인
2.  **Step Card (절차 안내)**:
    - `.step-grid`: CSS Grid를 활용한 2~3열 배치
    - `.step-card`: 순번(Step Number)이 강조되고 본문이 담기는 카드 형태
3.  **Typography & Table**:
    - `.guide-article`: 본문 가독성을 위한 최적의 자간, 행간, 폰트 크기 설정
    - `.guide-table`: 메뉴 경로 및 기능 설명을 위한 세련된 표 스타일 (Header 강조)
4.  **Media Component**:
    - `.image-wrapper`: 스크린샷 테두리와 그림자 효과
    - `.caption`: 이미지 하단 설명을 위한 작고 흐린 서체 스타일

### [Step 2] 메인 뷰어(index.html) 레이아웃 고도화
`index.html`의 콘텐츠 영역을 실제 '문서 도구'처럼 느껴지도록 개선하세요.

1.  **Breadcrumb (경로 표시)**:
    - 페이지 로드 시 상단에 `카테고리 > 중분류 > 소분류`가 자동으로 표시되도록 구현
2.  **Image Lightbox (확대 기능)**:
    - 가이드 내의 모든 `<img>` 태그 클릭 시, 별도 라이브러리 없이 검은 배경의 오버레이와 함께 이미지를 크게 확대해서 보여주는 기능 추가
3.  **Scroll UX**:
    - 본문 최대 너비를 `900px` 내외로 제한하여 시선 분산을 방지하고 중앙 정렬

### [Step 3] 표준 가이드 템플릿(template.html) 작성
향후 모든 가이드의 기준이 될 `pages/template.html`을 생성하세요. 
이 파일은 실제 가이드 작성 시 복사해서 사용할 수 있는 모든 CSS 컴포넌트의 샘플을 포함해야 합니다.

---

## 3. 검증 포인트 (Checklist)

- [ ] **가독성**: 텍스트 가독성(Line-height 1.6 이상)이 확보되었는가?
- [ ] **일관성**: 서로 다른 가이드 페이지를 로드해도 레이아웃이 깨지지 않는가?
- [ ] **반응형**: 모바일 기기에서 Step Card가 세로로 올바르게 정렬되는가?
- [ ] **확대**: 이미지 라이트박스가 부드럽게 작동하며, 닫기 버튼이 존재하는가?
