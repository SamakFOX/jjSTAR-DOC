star 관리

star 관리는 학생이 자주 사용하는 기본 학사/지원 메뉴를 한 곳에서 안내하는 통합 설명 페이지입니다. 실제 운영 시에는 아래 섹션별로 스크린샷 10-20장과 단계별 설명을 계속 추가하는 구조를 권장합니다.

이 페이지 역할

이 문서는 star 관리 중분류의 대표 소개 페이지입니다. 하위 소분류인 사제동행교수선택, 희망학과 선택, 교원 시간표 조회, 수강편람, 교육과정, 학사일정에 대한 공통 개요와 빠른 이동 링크를 제공합니다.

1. 포함된 하위 메뉴

  • 사제동행교수선택(신규)
  • 희망학과 선택
  • 교원 시간표 조회
  • 수강편람
  • 교육과정
  • 학사일정

2. 기본 사용 흐름

01

메뉴 진입

좌측 메뉴에서 star 학생지원 → star 관리를 선택합니다.

02

하위 기능 확인

중분류 페이지에서 소분류 목록과 주요 기능 요약을 먼저 확인합니다.

03

개별 상세 이동

필요한 항목을 선택하면 개별 상세 HTML 페이지를 fetch 방식으로 로드합니다.

3. 예시 화면 배치

이미지 01 - star 관리 메인 화면 캡처 영역

예시: star 관리 중분류 첫 화면과 하위 메뉴 진입 위치를 표시합니다.

이미지 02 - 소분류 드롭다운 노출 영역

예시: 중분류 페이지 진입 시 하위 소분류가 자동으로 펼쳐진 상태를 설명합니다.

이미지 03 - 공통 주의사항/자주 묻는 질문 영역

예시: 공통 안내 문구, 신청 전 확인사항, 담당 부서 연락처를 배치할 수 있습니다.

4. 작성 가이드

섹션 권장 내용
