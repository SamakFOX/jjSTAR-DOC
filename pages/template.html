JJSTAR 가이드 표준 템플릿

이 페이지는 JJSTAR 가이드 작성을 위한 모든 시각적 컴포넌트를 포함하고 있는 표준 템플릿입니다. 새로운 가이드를 작성할 때 이 파일의 구조를 복사하여 사용하세요.

안내: 가이드 문서의 가독성을 높이기 위해 적절한 위치에 강조 박스를 사용하세요.

1. 섹션 제목 (H2)

본문 텍스트는 .guide-article 클래스 내부에 작성합니다. 가독성을 위해 적절한 행간(1.6)과 자간이 적용되어 있습니다.

1.1 소제목 (H3)

하위 항목을 설명할 때 사용합니다.

2. 강조 박스 (Callouts)

정보: 일반적인 보충 설명이나 참고 사항을 전달할 때 사용합니다.
주의: 사용자가 실수하기 쉬운 부분이나 주의해야 할 사항을 강조할 때 사용합니다.
팁: 더 효율적으로 기능을 사용하는 방법이나 노하우를 공유할 때 사용합니다.

3. 단계별 카드 (Step Cards)

작업 절차를 안내할 때 Grid 시스템을 활용한 카드를 사용하세요.

1

메뉴 접속: 상단 메뉴에서 관리 페이지로 이동합니다.

2

정보 입력: 필수 항목을 모두 입력하고 확인 버튼을 누릅니다.

3

결과 확인: 성공 메시지가 출력되는지 확인합니다.

4. 표 (Tables)

메뉴 경로 안내나 기능 비교 시 세련된 표 스타일을 사용하세요.

구분 기능 설명 비고
등록 새로운 학생 정보를 시스템에 등록합니다. 필수 권한 필요
수정 기존 정보를 업데이트합니다. 로그 기록됨
삭제 데이터를 시스템에서 제거합니다. 복구 불가

5. 이미지 및 캡션 (Media)

스크린샷에는 그림자 효과와 테두리가 적용되며, 클릭 시 라이트박스로 확대됩니다.
